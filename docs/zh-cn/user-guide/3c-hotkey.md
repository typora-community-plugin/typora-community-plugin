# 快捷键设置


## 打开设置页

- 按下 <kdb>F1</kdb> 打开命令面板
- 输入 `设置` 找到命令
- 点击命令，在左侧导航栏选中 `快捷键` 标签页


## 添加快捷键

每个 Typora 社区插件提供的命令都列在该页面中。如果某个命令尚未绑定快捷键：

- 点击命令名称旁的 <span class="fa fa-plus primary"></span> 按钮
- 此时按钮文本变为「请按下快捷键...」
- 按下你想绑定的按键组合（例如 `Ctrl+Shift+K`）


## 移除快捷键

如果某个命令已绑定快捷键：

- 点击该快捷键标签右侧的 `<kbd>` 图标即可移除
- 移除后会重新出现 <span class="fa fa-plus primary"></span> 按钮，可以重新绑定


## 快捷键格式

快捷键以 `+` 连接修饰键和主键，例如 `Ctrl+Shift+K`。系统会自动归一化：

| 输入 | 归一化后（Windows/Linux） | 归一化后（macOS） |
|------|--------------------------|--------------------|
| `ctrl+k` | `Ctrl+K` | - |
| `meta+k`, `command+k`, `cmd+k` | `Win+K` | `Cmd+K` |
| `alt+k`, `option+k` | `Alt+K` | `Opt+K` |

命令分为 **全局**（对 Typora 所有区域生效）和 **编辑器**（仅焦点在编辑器内时生效）两种作用域，由插件自行定义。修改后的快捷键配置存储在插件的配置文件中，可在下次启动后自动加载。
