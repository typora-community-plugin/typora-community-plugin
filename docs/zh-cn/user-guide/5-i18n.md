# 国际化 (i18n)

Typora Community Plugin 提供语言切换功能，让用户可以自由选择插件的显示语言。


## 切换语言

在 Typora 的设置界面中，进入 **关于 (About)** 标签页，找到 **语言 (Language)** 选项：

- 点击下拉菜单，选择想要的语言
- 选择后插件会自动保存设置并生效

> 注意：切换语言后需要**重新启动 Typora** 才能完全生效。


## 支持的语种

目前支持的语种包括：

| 代号 | 名称 |
|------|------|
| `en` | 英语 |
| `zh-cn` | 简体中文 |
| `de` | 德语 |

更多语言可以提交 Pull Request 进行补充。


## 国际化文件结构

插件的国际化资源存放在 `packages/core/src/locales/` 目录下，使用 JSON 格式存储：

```
src/locales/
├── lang.en.json
├── lang.zh-cn.json
└── lang.de.json
```

每个语言文件中的键结构一致，对应 UI 各模块的文本：

```json
{
  "settingTabs": {
    "about": {
      "name": "About",
      "projectDesc": "A community plugin system for",
      "labelVersion": "Version",
      "buttonUpdate": "Update",
      "lang": "Language"
    }
  }
}
```

## 贡献翻译

如果你希望为插件添加新的语种翻译：

1. 新增一个语言文件，如 `lang.ja.json`
2. 参照已有语言文件，将所有键值翻译为目标语言
3. 提交 Pull Request

无需修改代码逻辑，只需添加对应的 JSON 翻译文件。
