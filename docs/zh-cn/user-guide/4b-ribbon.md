# 活动栏（Ribbon）

类似 VSCode 的活动栏，能够在不同侧栏面板间切换，包括内置面板（文件、搜索、大纲）和自定义面板（如插件 [标签][]）。

## 预览

![ribbon](../../assets/user/ribbon.jpg)

## 使用

### 切换侧栏面板

- **左键点击**图标可切换并显示对应的侧边栏面板
- 当前激活的图标会高亮显示

### 管理图标可见性

右键活动栏区域可以弹出菜单，在列表中对每个图标的可见性进行切换（显示/隐藏）。被隐藏的图标将不再显示在活动栏上。

### 调整图标顺序

拖动图标可上下调整其在活动栏中的位置，调整后会自动保存顺序。

## 设置按钮

活动栏底部有一个 **齿轮图标** 🛠️，提供以下快捷入口：

- 点击齿轮图标弹出菜单
  - **"应用设置"** — 跳转到 Typora 原生设置页面
  - **"插件配置"** — 打开社区插件的配置对话框（等同于快捷键 <kbd>Ctrl</kbd>+<kbd>.</kbd>）

## 配置

使用快捷键 <kbd>Ctrl</kbd>+<kbd>.</kbd> 打开"插件配置"对话框 → "外观"选项卡 → "高级选项"，勾选/反选 **"显示活动栏"** 可 显示/隐藏 整个活动栏。
