# typora-plugin-installer

Auto setup typora-community-plugin

## Install

### Windows

Runs with PowerShell:

```powershell
&'install.ps1'
```

Or with a custom installation path:

```powershell
&'install.ps1' -root <custom_install_path>
```

### macOS / Linux

Runs with Bash:

1. Open a terminal.
2. Navigate to the unzipped directory.
3. Run the valid command:

```bash
chmod +x install.sh
./install.sh
```

You may be prompted for your password (`sudo`) to modify Typora application files.
